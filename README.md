# Youtube React Clone
This is a simple youtube homepage clone, made with react.  

## Live Demo

- [Live Demo Link](https://clone-c5f62.web.app/)

## Built With
- React
- Material UI,

### Prerequisites

- node 14.9.0
- npm 6.14.8

### Install and Setup

- Clone repo
- cd into the cloned repo
- Open a terminal in the root of the app
- Run `npm install`
- Run `npm start`
- Open http://localhost:3000/ in your browser.
